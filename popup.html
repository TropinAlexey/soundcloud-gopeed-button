⬇ SoundCloud → Gopeed

Adds a Download button on SoundCloud track & playlist pages

Default is 9999 — Gopeed's built-in API port.
To check or change it: Gopeed → Settings → API → Port.

Set only if you enabled token auth in Gopeed → Settings → API → Token.
Leave empty for local Gopeed without auth.

Also required: install the gopeed-extension-soundcloud extension inside Gopeed so it can resolve SoundCloud URLs to audio streams.
